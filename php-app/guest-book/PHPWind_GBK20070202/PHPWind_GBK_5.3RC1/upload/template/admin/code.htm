功能介绍
营销计划定义：
　　PHPWind“论坛营销计划”是一套基于PHPWind软件产品下的营销产品，通过平台的整合营销优势，充分挖掘站长站点资源与潜力，为站长提取更多价值；站长在安装、使用PHPWind软件产品时，可自行选择开启或关闭，当您开启此项功能时，需注意下方提示信息；

平台优势：
　　“论坛营销计划”利用 PHPWind 平台把传统营销概念与论坛强大互动性相结合，充分挖掘论坛潜在价值， 解决网站经济来源问题，为站长解除后顾之忧；一方面借助 PHPWind 品牌优势，吸引广大商家的加盟，解决广告来源问题；一方面 PHPWind 拥有强大 的用户基础，论坛类型几乎覆盖所有行业，用户群集中，能满足广大商家的需求；自 PHPWind 5.0版本起， 系统后台集成了“论坛营销计划” 功能，站长只需要在后台进行相关设置即可加入“论坛营销计划”。

什么是通用密钥？
　　通用密钥是系统自动为每个网站生成的通用营销系统密钥（包括一对密钥和一个唯一识别码），用来做为网站的唯一标识；“通用密钥”将用于部分营销模块中，如内容营销模块；请保管好您的通用密钥，以防被他人利用。

提示信息：
　　当您开启部分需要读取页面的营销模块时，可能会耗损部分站点页面执行速度，这是由于您选择的营销模块特性及网络服务商的数据传送而产生的，正常情况下，影响并不大，如有特殊情况，导致站点访问效率明显下降是，可能由于服务商数据传送时出现网络问题，这时，我们建议您根据影响的程度考虑关闭部分营销模块，以确保站点的正常运行。

战略合作伙伴：
　　

通用密钥
“通用密钥”将用于部分营销模块中，如“内容营销模块”中要获得收益时需要用到此“通用密钥”
密钥一 $db_siteid
密钥二 $db_siteownerid
唯一识别码 $db_sitehash
内容营销模块
该模块无需注册，只要选择营销类型，点击“开启此模块”即可使用
注：内容营销模块一旦开启，系统就会自动把您应获得的收益记录到您的“通用密钥”中，您只需要使用“通用密钥”注册一个内容营销平台，即可查看并获得内容营销模块给您带来的收益，可能部分站点已经开启此模块，您的收益在此模块开启时就已经开始计算，查看获得收益步骤
营销类型 全站营销：在全站开启营销功能
帖子营销：只在帖子页面开启营销功能

站点营销模块
您需要先注册一个营销平台帐号才能开始使用站点营销模块，注册成功后登录营销平台，获得广告代码，利用论坛的广告管理功能，将广告代码投放到您的网站上
