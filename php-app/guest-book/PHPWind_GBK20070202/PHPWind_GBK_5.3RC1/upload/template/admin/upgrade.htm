会员组提升方案

    PHPWind 论坛系统提供了一套灵活的等级提升方案，可以根据综合积分来划分用户等级，设定一个好的等级提升方案能更好的体现论坛等级系统的价值

    综合积分：综合积分是按照会员组提升方案中设定的积分比率来计算的，可以根据论坛积分状况统计合理的设置会员组提升方案
    计算方法：各项积分*比率的和为综合积分的值

    如等级提升方案设置
    发贴数 : 1
    威望    : 0.5
    金钱    : -1
    其他    : 0

    综合积分计算方法如下：
    发贴数*1 + 威望*0.5 + 金钱*(-1)
