网上支付功能说明：
1. PHPWind 网上银行在线支付采用“贝宝网上银行系统”
2. 开通PHPWind 网上支付功能只需拥有一个“贝宝”帐号，并完成相关设置即可，不需要支付任何费用

使用网上支付相关说明：
1. 开启网上支付功能即被视为接受“PHPWind”相关协议，如有任何疑义，请关闭此功能
2. 支付过程出现问题请及时和PHPWind 官方取得联系
3. 因使用网站自身安全原因引起的非法支付PHPWind 不负任何责任


（一）开通贝宝支付步骤：

1. 注册一个“贝宝”帐号，注册贝宝帐号
2. 将“贝宝”帐号填写到论坛后台
    进入论坛后台 -> 网上支付设置 -> 网上支付 -> 贝宝信息 将您的贝宝帐号填写在“贝宝帐号”栏，提交！
3. 开启贝宝“即时付款通知”
    登录贝宝网站：进入 我的贝宝 -> 用户信息 -> 即时付款通知习惯设定 -> 编辑 开启“即时付款通知”并填写“通知URL”
    通知URL是 您的论坛网址 + '/paypal.php?verifycode=' + 贝宝公共密钥
    格式如：http://www.phpwind.net/paypal.php?verifycode=贝宝公共密钥
4. 完成以上步骤后即可开始使用PHPWind 网上支付功能

    获取贝宝公共密钥：进入 论坛后台 -> 网上支付设置 -> 网上支付 -> 贝宝信息 -> 贝宝公共密钥
    注：贝宝公共密钥是系统为您的网站自动生成的，请不要随意改动
